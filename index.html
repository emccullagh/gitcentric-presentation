Some basic commands

Checking out code

Git


						  $ git clone ssh://gitcentric:repo-name ~/dev

AccuRev


              $ accurev mkws -s stream-name -l ~/dev/workspace-name

Saving work

Git


$ git add .
$ git commit -m "JIRA-123: Commiting some changes"

AccuRev


							$ accurev keep -R -c "Keeping some changes" .

Sharing changes

Git


							$ git push

AccuRev


							$ accurev promote -P -c "JIRA-123: Promoting some changes"

Reverting changes

Git


							$ git checkout -- benchmarks.rb

AccuRev


							$ accurev purge benchmarks.rb
